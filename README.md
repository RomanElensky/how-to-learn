# Научиться учиться

## О проекте
На сайте собрана информация о разнообразных методах обучения с указанием полезных ссылок по теме. 

## О технологиях 
При создании сайта использовались такие технологии как:
1. Flex-box. 
2. Разнообразные виды позиционирвоания.
3. iframe.
4. Псевдоклассы. 
5. Анимации.
